# Språkstöd – plan

Målet är att kunna publicera samma material på flera språk utan att ändra innehåll eller struktur.

## Struktur
- Svenska original:
  - `docs/onboarding-sv.md`
  - `docs/handbok-sv.md`
- Kommande översättningar:
  - `docs/onboarding-en.md`
  - `docs/handbook-en.md`
  - `docs/onboarding-de.md`
  - `docs/handbook-de.md`

## Riktlinjer för översättning
- Behåll rubriknivåer och avsnittsordning
- Behåll bild‑placeholders och filnamn för enkel återanvändning
- Använd fackspråk som är vanligt inom släktforskning
- Översätt inte filvägar eller kodblock
